# 愿景落地

HnsX 的愿景是 **Harness as a Service**：让企业能够安全、可控、可评估地驾驭最强 Agent。

## 产品形态

1. **SDK**：开发者自己定义 Agent，再把它部署到 Harness 上运行。
2. **平台**：在平台上管理 Harness 和 Agent，包括部署、监控、评估等。

## 我们不做的事

- ❌ 不造 Agent 底座
- ❌ 不做低代码拖拽编排
- ❌ 不替代 Claude Code / Codex 等专业 Agent

## 我们只做一件事

让**业务工程师 / Harness 设计师**能够用声明式的方式，把最强 Agent 装进企业需要的约束里。
